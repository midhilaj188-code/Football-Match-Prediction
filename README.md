# ⚽ Football Match Prediction App

This project is a simple Machine Learning web application built using Streamlit.  
It predicts whether the home team will win a football match based on match statistics.

---

## 🚀 Features

- Predicts match outcome (Win / Lose)
- User-friendly interface using Streamlit
- Input validation for realistic football data
- Machine Learning model using Random Forest

---

## 📊 Input Features

The model uses the following features:

- Halftime Home Goals
- Halftime Away Goals
- Total Goals
- Goal Difference

---

## 🤖 Model Details

- Algorithm: Random Forest Classifier
- Target: Home Team Win (1) or Lose (0)
- Trained on historical football match dataset
